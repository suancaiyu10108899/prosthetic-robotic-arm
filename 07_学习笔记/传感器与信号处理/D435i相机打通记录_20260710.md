## 六、注意事项

- SDk 安装后必须**重启电脑**才能生效
- USB 线必须是 **USB 3.0**（蓝色接口），普通充电线不行
- Viewer 和 Python 脚本**不能同时使用**相机（相机只能被一个进程占用）
- `rs-enumerate-devices.exe` 和 `realsense-viewer.exe` 不在系统 PATH 中，需要用完整路径或从开始菜单启动
