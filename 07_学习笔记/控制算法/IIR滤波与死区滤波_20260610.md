# IIR 低通滤波与死区滤波

> **日期**：2026-06-10
> **主题**：分析 `sketch_feb3a.ino` 中两个滤波器
> **来源**：项目固件代码

---

## 一、为什么需要滤波？

Bend Labs 弯曲传感器输出原始角度数据，包含：
- **高频噪声**（电路噪声、机械振动）→ 用低通滤波消除
- **小幅度抖动**（传感器静止时微小波动）→ 用死区滤波消除

## 二、二阶 IIR 低通滤波器 `signal_filter()`

### 代码
```cpp
void signal_filter(float * sample) {
    static float filter_samples[2][6];

    for(uint8_t i=0; i<2; i++) {
        filter_samples[i][5] = filter_samples[i][4];
        filter_samples[i][4] = filter_samples[i][3];
        filter_samples[i][3] = (float)sample[i];
        filter_samples[i][2] = filter_samples[i][1];
        filter_samples[i][1] = filter_samples[i][0];

        // 20 Hz 截止频率 @ 100 Hz 采样率
        filter_samples[i][0] =
            filter_samples[i][1] * (0.36952737735124147f)
          - 0.19581571265583314f * filter_samples[i][2]
          + 0.20657208382614792f * (filter_samples[i][3]
            + 2 * filter_samples[i][4] + filter_samples[i][5]);

        sample[i] = filter_samples[i][0];
    }
}
```

### 工作原理

这是一个 **二阶无限脉冲响应（IIR）低通滤波器**，设计参数：

| 参数 | 值 |
|------|-----|
| 类型 | Butterworth 低通（推测） |
| 采样率 (Fs) | 100 Hz |
| 截止频率 (Fc) | **20 Hz** |
| 阶数 | 2 |

### 为什么要 20Hz？

人体关节运动的频率通常 **<5 Hz**（即使是跑步，手臂摆动也不超过3-4Hz）。20Hz 截止频率足以保留所有有用运动信号，同时滤除 50Hz 工频干扰和高频噪声。

### 差分方程

滤波器差分方程形式：
```
y[n] = b0 × x[n] + b1 × x[n-1] + b2 × x[n-2]
     - a1 × y[n-1] - a2 × y[n-2]
```

对应代码中的系数：
| 系数 | 值 | 含义 |
|:---:|-----|------|
| b₀ | 0.2066 | 当前输入权重 |
| b₁ | 2×0.2066 = 0.4132 | 前一输入权重 |
| b₂ | 0.2066 | 前二输入权重 |
| a₁ | -0.3695 | 前一出反馈 |
| a₂ | 0.1958 | 前二出反馈 |

### 缓冲管理技巧

用滚动缓冲区避免频繁 `memmove`：
```
[0] = 当前输出
[1] = 前一次输出
[2] = 前两次输出
[3] = 当前输入
[4] = 前一次输入
[5] = 前两次输入
```
每次新数据到达，只需把所有值**右移一位**，再用差分方程算 `[0]`。

## 三、死区滤波器 `deadzone_filter()`

### 代码
```cpp
void deadzone_filter(float * sample) {
    static float prev_sample[2];
    float dead_zone = 0.5f;  // 0.5° 死区

    for(uint8_t i=0; i<2; i++) {
        if(fabs(sample[i] - prev_sample[i]) > dead_zone)
            prev_sample[i] = sample[i];   // 变化>0.5° 则更新
        else
            sample[i] = prev_sample[i];   // 否则保持上一次
    }
}
```

### 原理

| 变化量 | 行为 |
|:-----:|------|
| > 0.5° | 更新输出 = 新值 |
| ≤ 0.5° | 保持上次值不变 |

**效果**：角度变化 <0.5° 时输出完全静止，消除传感器静止时的"抖动"。

### 为什么是 0.5°？

这个值可以根据实际情况调整：
- 太小 → 不起作用
- 太大 → 小动作丢失
- **0.5°** 是针对 Bend Labs 传感器经验值

## 四、两条滤波器的串联

```
原始角度数据
    ↓
signal_filter()     ← 先低通滤高频噪声
    ↓
deadzone_filter()   ← 再死区去小抖动
    ↓
newData = true      ← 触发舵机更新
```

**注意**：死区滤波器**只在双轴传感器时运行**（`adsType == ADS_TWO_AXIS`），单轴传感器跳过 `signal_filter()` 直接进入死区。

---

## 五、要点总结

| 滤波器 | 类型 | 消除 | 参数 |
|--------|------|------|------|
| `signal_filter` | IIR 低通 (二阶 Butterworth) | 高频噪声 | Fc=20Hz @ Fs=100Hz |
| `deadzone_filter` | 非线性阈值 | 静止抖动 | 阈值=0.5° |

**记住**：滤波顺序是先低通后死区，不可颠倒。先平滑再截断，才能得到干净信号。
