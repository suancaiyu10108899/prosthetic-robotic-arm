## 二、三块板子

| 编号 | 位置 | 核心芯片 | 功能 | 编程/调试方式 |
|:---:|------|----------|------|-------------|
| ① | 电池箱 | Arduino (待确认型号) | 接收蓝牙→解析指令→发有线信号 | Arduino IDE + 有线 USB |
| ② | 电池箱 | （电压转换芯片） | 电压转换 | 无需编程 |
| ③ | 机械臂上 | STM32 (待确认) | 信号→舵机/电机控制 | 塔克 DAP 无线调试器 |

## 三、信号流

```
蓝牙指令
  → ① Arduino 蓝牙板 接收 + 解析
  → ① (UART/SPI/I2C) 有线传给 ③ 控制板
  → ③ STM32 生成 PWM
  → 舵机动作
```

## 六、塔克 DAP 无线调试器要点

| 参数 | 值 |
|------|-----|
| 芯片 | ESP32-S3 模组 |
| 协议 | SWD + JTAG |
| 距离 | ≥30m |
| 速度 | 接近有线 STLink V2 |
| 模式 | 有线 / 无线发射 / 无线接收（按键切换） |
| 虚拟串口 | 支持 |
| 系统 | Windows/Linux/Mac |
