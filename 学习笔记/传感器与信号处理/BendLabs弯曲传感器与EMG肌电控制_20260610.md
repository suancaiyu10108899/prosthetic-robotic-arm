# Bend Labs 弯曲传感器与 EMG 肌电控制

> **日期**：2026-06-10
> **主题**：本项目使用的两种传感器方案
> **来源**：固件代码分析 + 主控板 PCB 分析

## 一、为什么要两种传感器？

本项目同时具备两种控制方案，这是学术竞赛级假肢的常见做法：

1. **弯曲传感器** — 测量**机械关节角度**（物理量）
2. **EMG 肌电信号** — 测量**人体肌肉电信号**（生物量）

两者可以独立工作，也可以**融合**提高控制精度。

---

## 二、Bend Labs 弯曲传感器

### 原理

Bend Labs 的传感器基于**电容变化**检测弯曲角度。传感器是一根柔性带状物，弯曲时内部电容变化 → 转换为角度值。

### 型号

| 型号字符串 | 含义 |
|-----------|------|
| `ADS_ONE_AXIS` | 单轴传感器：测量弯曲角度 + 拉伸量 |
| `ADS_TWO_AXIS` | 双轴传感器：测量两个正交方向的弯曲角度 |

### 关键参数

| 参数 | 值 | 来源 |
|------|-----|------|
| 角度范围 | ±250° | 从固件映射公式反推 |
| 输出精度 | 0.5° | 死区滤波器阈值 |
| 采样率 | 100 Hz | IIR 滤波器设计参数 |
| 通信方式 | BLE 5.0 无线 | `bluefruit.h` 库 |

### 数据格式

每次 BLE 通知 8 字节 = 两个 IEEE 754 单精度浮点数：
```
[X轴角度(float) | Y轴角度(float)]
```

### 在假肢中的位置

最典型的安装位置：**假肢手指关节**的柔性部分。手指弯曲 → 传感器弯曲 → 输出角度 → 控制对应的电机/舵机。

## 三、EMG 肌电信号

### 什么是 EMG？

**肌电图 (Electromyography)** — 肌肉收缩时产生的微弱电信号（微伏～毫伏级）。

### 在 PCB 中的证据

主控板 `core.PcbDoc` 上有两个专用接口：

| 位号 | 名称 | 连接器类型 |
|:---:|------|-----------|
| P4 | EMG-1 | 3P-1.0mm FPC |
| P5 | EMG-2 | 3P-1.0mm FPC |

**双通道** EMG 意味着可以同时采集**两组拮抗肌**（如屈肌+伸肌）。

### 工作原理链

```
人脑 → 运动神经元 → 肌肉纤维收缩 → 微弱电位变化 (EMG)
    ↓
电极贴片 → 差分放大 → 带通滤波 (20-500Hz)
    ↓
整流+平滑 → 包络检波 → ADC 采样 (STM32U575)
    ↓
特征提取 → 控制算法 → 执行器 (舵机/电机)
```

### 为什么需要 STM32U575？

- EMG 信号处理需要**较高的采样率**（≥1kHz 每通道）
- 实时信号处理需要 **DSP 指令集**（STM32U5 有 MVE 向量扩展）
- 多通道同步采样需要**硬件定时器精确控制**

nRF52840 负责蓝牙通信，STM32U575 负责实时信号处理——**分工明确**。

## 四、传感器融合的意义

| 传感器 | 优势 | 劣势 |
|--------|------|------|
| 弯曲传感器 | 直接测量物理位置，精确 | 只能测已安装关节 |
| EMG | 直接读取人的意图 | 噪声大，个体差异大 |

**融合方案**：EMG 提供**意图预测**（"用户想抓"），弯曲传感器提供**实际状态反馈**（"手指到了 30°"），两者结合实现闭环控制。

## 六、要点总结

1. 两个传感器解决两种需求：**机械反馈** (弯曲传感器) + **人的意图** (EMG)
2. 弯曲传感器走 BLE → nRF52840 → 舵机 PWM
3. EMG 走模拟信号 → STM32U575 ADC → 算法处理
4. 两个传感器可以融合做闭环控制
5. 目前固件只实现了弯曲传感器方案，EMG 端还在 PCB 预留阶段
