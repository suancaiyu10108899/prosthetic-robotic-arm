## 项目简介

本项目旨在研究/开发假肢机械臂系统，涉及机械结构设计、电子控制、嵌入式开发与上位机通讯等方向。

## 文件命名规范

- **格式**：`[项目简写]_[部件名称]_V[版本号].[后缀]`
- **示例**：`ARM_ElbowJoint_V3.SLDPRT`、`ARM_Finger_V2.stl`

## 版本管理说明

- 本项目使用 **Git** 进行版本管理
- **大文件（>10MB）** 不纳入 Git 追踪，详见 `.gitignore`
- STL 扫描模型、大尺寸 SolidWorks 文件请存至 `01_机械结构/STL/扫描模型/` 或 `01_机械结构/STL/打印模型/`
