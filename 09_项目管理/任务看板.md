## 📝 备注

- nRF52840 UART：D0(TX)=P0.25, D1(RX)=P0.24 — 通过 `Serial1` 使用
- 烧录命令：`python -m platformio run --project-dir=d:\Dev\arm-ble --target upload`
- 串口监视：`python -m platformio device monitor --project-dir=d:\Dev\arm-ble --baud 115200`
- 手套一键启动：WSL 内 `bash D:\Dev\data-collection\scripts\wsl_gui_launcher.sh`
- GitHub (固件): https://github.com/suancaiyu10108899/arm-ble-firmware
