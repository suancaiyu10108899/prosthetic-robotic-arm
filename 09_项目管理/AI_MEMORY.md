## 4. 重要约定与规则

1. **STL/SolidWorks 大文件**：不入 Git（`.gitignore` 已配置），不会被 AI 读取
2. **文件命名规范**：
   - **你自己画的零件**：`【打印件/加工件/铝方管】零件名.后缀`（如 `【打印件】机械臂盖板.SLDPRT`）
   - **老师/他人发来的文件**：保留原名（如 `内腔.stl`、`敏姐外腔-小周改.stl`）
   - **版本号**：如有多版，加 `_V2`、`_V3` 后缀（如 `【打印件】机械臂盖板_V2.STL`）
3. **打印材料**：未来工场8200pro树脂（光固化） + 尼龙件
4. **设计笔记**：每次重大设计变更在对应模块的 `设计笔记/` 记录
5. **Git 只追踪文档，不追踪大文件**
