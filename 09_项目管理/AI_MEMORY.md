### 电控系统架构（已确认）

```
蓝牙信号源 (手机/手柄)
    │ BLE
    ▼
┌─ 电池模块 (电池箱) ────────────────┐
│  ① Adafruit Feather nRF52840 Express   │  ← 🔴 当前任务：BLE 调试验证
│     (nRF52840, MicroUSB 供电+编程)     │
│  ② 电压转换板                         │  ← 纯硬件，无需编程
└──────────┬──────────────────────────┘
           │ 有线信号 (UART/SPI/I2C)
           ▼
┌─ 机械臂控制板 ──────────────────────┐
│  ③ STM32/cybathlon   ← 发PWM        │  ← 用塔克DAP无线调试
└────────────────────────────────────┘
```

- **板①**：**Adafruit Feather nRF52840 Express**（nRF52840 芯片，已实物确认丝印，✅ BLE 广播+连接+收发全链路验证通过，MAC: D3:52:88:3A:06:27，NUS UID: 6E400001-B5A3-F393-E0A9-E50E24DCCA9E）

- **板③**：STM32 控制板，用塔克 DAP（ESP32-S3, 30m）无线调试（在学长处）

1. **STL/SolidWorks 大文件**：不入 Git（`.gitignore` 已配置），AI 不读取内容
2. **文件命名规范**：
   - 你自己画的零件：`【打印件/加工件/铝方管】零件名.后缀`
   - 老师/他人发来的文件：保留原名

3. **打印材料**：未来工场8200pro树脂（光固化）+ 尼龙件
4. **开发代码与文档分离**：代码在 `D:\Dev\arm-ble\` 和 `D:\Dev\arm-ble-gui\`（纯英文路径），文档在 `d:\假肢机械臂\`
