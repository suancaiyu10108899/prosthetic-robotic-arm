### 电控系统架构（已确认）

```
蓝牙信号源 (手机/手柄)
    │ BLE
    ▼
┌─ 电池模块 (电池箱) ────────────────┐
│  ① Adafruit Feather nRF52840 Express   │  ← 🔴 当前任务：待烧录验证
│     (nRF52840, MicroUSB 供电+编程)     │
│  ② 电压转换板                         │  ← 纯硬件，无需编程
└──────────┬──────────────────────────┘
           │ 有线信号 (UART/SPI/I2C)
           ▼
┌─ 机械臂控制板 ──────────────────────┐
│  ③ STM32/cybathlon   ← 发PWM        │  ← 用塔克DAP无线调试
└────────────────────────────────────┘
```

- **板①**：**Adafruit Feather nRF52840 Express**（nRF52840 芯片，已实物确认丝印）
- **板②**：电压转换，纯硬件，无需编程
- **板③**：STM32 控制板，用塔克 DAP（ESP32-S3, 30m）无线调试
