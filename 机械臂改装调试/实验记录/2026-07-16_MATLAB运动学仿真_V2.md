## 2. 已确认机构

- 世界原点：袖筒与第一旋转基座连接处；袖筒固定
- 主链 3R：X-Y-X
  - q1：点 `(0,0,0)`，轴 `(1,0,0)`，±180°
  - q2：点 `(50,0,0)`，轴 `(0,1,0)`，-90°~15°
  - q3：点 `(80,0,0)`，轴 `(1,0,0)`，±180°
- 夹爪是平行四边形连杆与齿轮耦合的单自由度机构
- PDF 预实验限位：g=0°~90°

## 3. 夹爪方向校准

第一版使用 PDF 表面符号时，STL 指腹出现 X 型交叉。按实际装配显示，修正为：

```text
qA=+g
qB=-g
qTipA=-g
qTipB=+g
```

该约束仍满足两侧指腹总姿态补偿为零，保持平行。

## 4. TCP 定义

每侧从 Fingertip 旋转轴沿该指腹局部 +X 前移 40 mm，得到 Contact A/B；TCP 是两接触参考点中点。

零位：

```text
Contact A = [227, 8, 0] mm
Contact B = [227,-8, 0] mm
TCP       = [227, 0, 0] mm
```

## 5. 雅可比

- 使用最小主动坐标 `q=[q1;q2;q3;g]`
- 建立基坐标系表达的 6×4 几何雅可比 `[Jv;Jw]`
- 主链列采用 `a_i × (pTCP-p_i)` 和 `a_i`
- 平行四边形使 g 不改变 TCP 姿态，故第 4 列角速度为 0
- 修正夹爪方向后：

```text
x_TCP^G = rootX + linkX*cos(g) - linkY*sin(g) + contactX
dx/dg  = -linkX*sin(g) - linkY*cos(g)
```

V1 旧方向曾通过中心差分验证；V2 提供“验证雅可比”按钮，用于修正方向后的在线验证。
