# 机械臂控制

> **所属项目**：机械臂改装调试
> **状态**：🟡 MATLAB 仿真 + 实物控制上位机开发
> 最后更新：2026-07-16

---

## 目标

对 3 自由度腕部 + 夹爪的机械臂进行运动学建模、MATLAB 仿真求解正逆解、STL 装配体转 URDF，最终在 Windows 上通过 PC 直连 4 个 Feetech 舵机实现实物控制。

## 设备清单

| 设备 | 型号/标识 | 数量 | 状态 |
|------|------|:--:|:--:|
| 舵机（大） | Feetech HL-3625-C001 | 2 | 🟡 待控制打通 |
| 舵机（小） | Feetech HL-3612-C001 | 2 | 🟡 待控制打通 |
| 机械臂本体 | 3-DOF 腕部 + 夹爪 | 1 套 | 🟡 STL 模型就绪 |

## 关键参数

- **自由度**：3-DOF 腕部（wrist_roll + wrist_body 俯仰/偏转）+ 1 夹爪开合 = 4 舵机
- **舵机型号**：HL-3625-C001 ×2、HL-3612-C001 ×2
- **舵机协议**：Feetech SCS 串口协议（半双工 UART，1Mbps）
- **控制方式**：PC 直连（USB 转 TTL），无单片机
- **仿真工具**：MATLAB（正/逆运动学求解）
- **实物控制**：Python 上位机（PyQt）
