## 关键参数

- **自由度**：3-DOF 腕部（wrist_roll + wrist_body 俯仰/偏转）+ 1 夹爪开合 = 4 舵机
- **舵机型号**：HL-3625-C001 ×2、HL-3612-C001 ×2
- **舵机协议**：Feetech SCS 串口协议（半双工 UART，1Mbps）
- **控制方式**：PC 直连（USB 转 TTL），无单片机
- **仿真工具**：MATLAB（正/逆运动学求解）
- **实物控制**：Python 上位机（PyQt）

## 当前数学模型

独立广义坐标：`q=[q1;q2;q3;g]`。主腕部正运动学：

```text
T0G = Rx(q1) Tx(50) Ry(q2) Tx(30) Rx(q3)
```

夹爪受约束角：

```text
[qA;qB;qTipA;qTipB] = [1;-1;-1;1] g
```

理想平行四边形下 TCP 姿态与 Gripper 坐标系一致。TCP 位置雅可比包含夹爪第 4 列；`q3` 改变 TCP 姿态但因 TCP 在滚转轴线上而不改变其位置。
