# 机械臂改装调试

> 🟡 **启动阶段** — 为机械臂集成触觉传感器、灵巧手，实现多模态感知与控制
>
> 最后更新：2026-07-16

---

## 项目目标

将现有机械臂本体升级为具备 **触觉感知 + 灵巧手操作** 的智能平台：

1. **触觉传感器**：SPI→USB（FT232H）打通，PC 端实时读取触觉数据
2. **LinkerHand 灵巧手**：接口方案调研 → 通信打通 → 控制集成
3. **机械臂本体**：改装方案设计，三设备统一 PC 端控制

## 三条线并行
