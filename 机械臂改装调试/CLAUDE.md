## 技术参数速查

- **FT232H 芯片**: FTDI FT232HQ, USB 2.0 Hi-Speed → SPI / I2C / JTAG / UART / FIFO
- **触觉传感器接口**: SPI（SCK / MOSI / MISO / CS）
- **舵机协议**: Feetech SCS 串口（半双工 UART，1Mbps）
- **机械臂自由度**: 3-DOF 腕部 + 1 夹爪 = 4 舵机
