## 四、Python 版本管理方案

三个独立环境，互不冲突：

| 环境 | Python | 位置 | 用途 |
|------|:--:|------|------|
| 系统默认 | 3.11 (`D:\Python-3.11.9\`) | 全局 | VS Code / Cline / D435i / NOKOV |
| sparqi-env | 3.10 虚拟环境 | `D:\Dev\sparqi-env\.venv` | ResearchKit SDK（Spar Qi 手环） |
| WSL2 内部 | 3.10 | WSL2 独立安装 | TouchGlove SDK（手套） |

调用方式：
```powershell
# 日常
python xxx.py                            # 系统 Python 3.11

# Spar Qi 项目
D:\Dev\sparqi-env\.venv\Scripts\python xxx.py

# 手套项目 (WSL2 终端内)
python3 xxx.py
```

## 七、关键风险与缓解

| 风险 | 概率 | 缓解 |
|------|:--:|------|
| WSL2 CUDA 不可用 | 低 | WSL2 官方支持 CUDA，已确认驱动 560.94 兼容 |
| ResearchKit wheel 在 Python 3.10 虚拟环境中安装失败 | 低 | 前人已验证，我们用的是同一 wheel |
| TouchGlove SDK 不支持 WSL2 的串口 | 中 | 可能需要配置 USB 直通→WSL2 (`usbipd`) |
| NOKOV 服务器 IP 变化 | 中 | 前人硬编码 `10.1.1.198`，需要向学长确认 |
| 手套数据量过大导致 D 盘满 | 低 | 197GB 可存 ~50 次完整实验，加上移动硬盘备份 |
